# Frontend


## Основные папки

- `src/app` — роутер
- `src/components` — layout и общие UI-компоненты
- `src/features/auth` — логин, регистрация, auth context
- `src/features/projects` — api и типы проектов
- `src/lib` — api client, storage, query client
- `src/pages` — страницы
- `src/theme` — MUI theme
